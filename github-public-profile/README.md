# Kim Liao

![Photo of Kim Liao](9B1AA89E-0809-4636-AA62-55BAE3A88876.JPG)

Kim Liao is a senior in the Data Visualization program at the University of Washington Bothell. He is interested in using data to improve systems, solve problems, and make information easier for people to understand. His interest in data and visualization comes from seeing how better systems can improve real experiences and reduce mistakes in everyday settings. He has experience working with tools like Tableau, Excel, R, and Processing, and he enjoys creating visualizations that are clear, practical, and meaningful. As a future data visualization professional, he hopes to build work that helps people understand patterns, communicate ideas more effectively, and make better decisions.
